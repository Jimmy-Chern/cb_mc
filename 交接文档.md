# 可转债MC定价自动交易系统 — 交接文档

## 一、系统概述

浙大A800服务器每天15:30自动运行蒙特卡洛定价模型（5000条路径，复现论文 arXiv:2409.06496），对全市场可转债排序，选出理论价最被低估的Top10，加密推送到GitHub Gist。

Windows笔记本上的QMT每天14:50从Gist拉取信号，解密后自动调仓。

```

    └→ 覆盖bond_zh_cov的CB现价

[3] FTShare MCP ft_stock_candlesticks_batch
    └→ 321只正股当日收盘价 (~33次API, ~5秒)
    └→ 增量写入 parquet 缓存 (为明天波动率准备)

[4] 波动率计算
    └→ 从 ~/chenjunming/quant/cache/st_fresh/*.parquet 读取252天日线
    └→ 712只正股覆盖, 年化波动率 σ

[5] GPU MC定价
    └→ 5000条路径 x 324只转债, A800 GPU加速
    └→ LSM后向归纳 + 多区间回归

[6] 排序选股 + AES加密 + 推送Gist
    └→ cb_signal.json (加密, 需密码336018)
    └→ cb_log.json (明文日志)
```

### 4.2 QMT策略代码

```python
# ===== 可转债MC定价策略 QMT客户端 =====
import requests, json, hashlib, base64, os
import pyaes

GIST_URL = "https://gist.githubusercontent.com/Jimmy-Chern/4f3d9550b2c5e874019272b025a58408/raw/cb_signal.json"
PASSWORD = "336018"

def decrypt_aes(encrypted, password):
    """AES-CBC解密 (pyaes, Python 3.6兼容, bytes padding)"""
    key = hashlib.sha256(password.encode()).digest()
    raw = base64.b64decode(encrypted)
    iv, ct = raw[:16], raw[16:]
    cipher = pyaes.AESModeOfOperationCBC(key, iv=iv)
    plain = b''
    for i in range(0, len(ct), 16):
        plain += cipher.decrypt(ct[i:i+16])
    pad_len = plain[-1]
    return json.loads(plain[:-pad_len].decode('utf-8'))

def get_signal():
    try:
        resp = requests.get(GIST_URL, timeout=15)
        signal = decrypt_aes(resp.json()["encrypted"], PASSWORD)
        print(f"[{signal['date']}] Top10: {[p['code']+p['name'] for p in signal['positions']]}")
        return [p['code'] for p in signal['positions']]
    except Exception as e:
        print(f"信号获取失败: {e}")
        return []

def init():
    global g_target_codes
    g_target_codes = get_signal()

def handlebar(ContextInfo):
    current_time = ContextInfo.get_current_time()
    if current_time.strftime('%H:%M') != '14:50':
        return

    global g_target_codes
    target = set(g_target_codes)
    current = set()
    vol_map = {}
    for code in ContextInfo.get_stock_code_list():
        if code.startswith(('11','12')):
            vol = ContextInfo.get_volume(code)
            if vol > 0:
                current.add(code)
                vol_map[code] = vol

    # 卖出不在目标的
    for code in current - target:
        vol = vol_map.get(code, 0)
        if vol > 0:
            print(f"  卖出 {code} x{vol}")
            passorder(24, 1101, ContextInfo.accID, code, 5, 0, vol, '', 1, '', ContextInfo)

    # 买入新增的
    to_buy = target - current
    if to_buy:
        cash = ContextInfo.get_account_asset('STOCK').m_dCash * 0.95
        per = cash / len(to_buy)
        for code in to_buy:
            price = ContextInfo.get_full_tick(code).m_nMatch
            if price > 0:
                vol = int(per / price / 10) * 10
                if vol >= 10:
                    print(f"  买入 {code} x{vol}")
                    passorder(23, 1101, ContextInfo.accID, code, 5, price, vol, '', 1, '', ContextInfo)

    # 收盘后获取明天的信号
    g_target_codes = get_signal()
```

**无需修改任何代码**，粘贴到QMT策略编辑器直接运行。

## 五、信号格式

解密后的JSON结构：
```json
{
  "date": "2026-07-15",
  "time": "17:35:22",
  "n_bonds_total": 1034,
  "n_bonds_priced": 324,
  "n_positions": 10,
  "buy": ["127079", "118071", "110092", ...],
  "positions": [
    {
      "code": "127079",
      "name": "华亚转债",
      "market_price": 100.0,
      "model_price": 272.4,
      "discount": 172.4,
      "stock_price": 22.5,
      "conv_price": 15.3,
      "volatility": 0.45
    },
    ...
  ]
}
```

关键字段：
- `buy` — 要持有的10个转债代码（已按折价率排序）
- `discount` — 折价率，(模型价-市场价)/市场价。正值=被低估
- `model_price` — MC模型理论价
- `market_price` — 当日实盘市场价

## 七、健康监控

### 7.1 查看运行日志

```python
import requests
log = requests.get("https://gist.githubusercontent.com/Jimmy-Chern/4f3d9550b2c5e874019272b025a58408/raw/cb_log.json").json()
print(f"最近运行: {log['run_at']}, 耗时{log['duration_s']}秒, 错误{len(log['errors'])}个")
```

### 7.2 日志告警规则

| 日志内容 | 含义 | 动作 |
|----------|------|------|
| `bond_zh_cov拉取失败` | 数据源挂了 | ❌ 今日不交易 |
| `errors > 0` | 有异常 | ⚠️ 检查详情再决定 |
| `errors == 0` | 正常 | ✅ 可以交易 |
| `n_bonds_priced < 50` | 可定价转债太少 | ⚠️ 数据可能大面积异常 |

### 7.3 QMT里加健康检查

在QMT策略的handlebar开头加：
```python
# 14:00先检查日志
if current_time.strftime('%H:%M') == '14:00':

        return
```

## 十、常见问题

**Q: 第一天没有信号怎么办？**
让服务器管理员手工跑一次：`cd /home/xujiayang2/chenjunming/cb_mc && .venv/bin/python push_signal.py`

**Q: 信号里折价100%+正常吗？**
正常。MC模型价≈债底≈100-120元，但市场可能只给70-80元（深度虚值转债）。这类债赌下修或正股暴涨。

**Q: GitHub被墙怎么办？**
服务器用代理推送，QMT端如果拉不到，需要开代理或换国内文件托管。

**Q: 想换选股数量？**
服务器上改 `push_signal.py` 里的 `N_POS = 10`，QMT里不需要改。
