# 项目迁移文档 — 换机器重新组装

本项目已拆成三部分存储，换机器后按本文档重新组装。

## 组装清单

| 部分 | 存储位置 | 内容 |
|------|----------|------|
| 代码 | GitHub `Jimmy-Chern/cb_mc` | 全部Python代码+文档 |
| 数据 | HuggingFace `moonmonster/cb_mc_data` (私有dataset) | 历史行情缓存 (39MB压缩) |
| 环境 | `requirements_frozen.txt` (在代码仓库里) | pip依赖清单 |

## 一、拉取代码

```bash
git clone https://github.com/Jimmy-Chern/cb_mc.git
cd cb_mc
```

## 二、拉取数据

数据在HuggingFace私有dataset，需要token：

```bash
pip install huggingface_hub
export HF_TOKEN=<你的HF_token>
# 如果外网需要代理:
export https_proxy=http://127.0.0.1:7899

python3 -c "
from huggingface_hub import hf_hub_download
f = hf_hub_download(repo_id='moonmonster/cb_mc_data',
                    filename='cb_mc_data.tar.gz',
                    repo_type='dataset',
                    token='<你的HF_token>')
print('下载到:', f)
"
```

解压到正确位置（数据必须放在 `~/chenjunming/quant/` 下）：

```bash
mkdir -p ~/chenjunming/quant
tar xzf cb_mc_data.tar.gz -C ~/chenjunming/quant/
```

解压后目录结构：
```
~/chenjunming/quant/
├── cache/
│   ├── cb_hist.pkl        # 212只转债历史日线 (2020-2026)
│   ├── st_hist.pkl        # 对应正股历史日线
│   └── st_fresh/          # 924只正股252天日线 (parquet)
│       ├── 600000.parquet
│       └── ...
└── data/
    └── cb_daily/          # 1005只转债日线 (parquet)
        ├── 110001.parquet
        ├── manifest.pkl
        └── ...
```

## 三、重建Python环境

```bash
cd cb_mc
python3.11 -m venv .venv
source .venv/bin/activate

# CUDA 12.4 版PyTorch必须单独装
pip install torch==2.6.0 torchvision==0.21.0 torchaudio==2.6.0 --index-url https://download.pytorch.org/whl/cu124

# 其余依赖
pip install -r requirements_frozen.txt
```

核心依赖：
```
torch==2.6.0+cu124   # GPU加速MC
numpy, pandas        # 数据处理
akshare==1.18.64     # A股数据源
pyarrow==25.0.0      # parquet读写
pyaes==1.6.1         # AES加密(QMT兼容)
cos-python-sdk-v5    # 腾讯云COS上传
requests, tqdm       # HTTP+进度条
huggingface_hub      # 数据备份
```

## 四、填回密钥

代码里的密钥已用占位符替换，需重新填：

**push_to_cos.py** (第11-12行):
